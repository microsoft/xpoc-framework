XPOC static sample pages

This is a collection of sample static client-side HTML pages that can be used to test the XPOC framework. These are available in the XPOC GitHub project.

Note: the viewer and verifier pages might not be able to retrieve external pages (including manifests) depending on the browser's security and the server's CORS settings. The more powerful server-backend sample pages can be used to prevent these issues.

  • Manifest creator: create a XPOC manifest by entering a user's accounts on the most popular platforms.
  • Manifest editor: create and update XPOC manifests, adding platform accounts and content items.
  • CSV-to-manifest converter: transform comma-separated-value (.csv) files listing platform accounts and content items into a XPOC manifest.
  • Manifest viewer: retrieve and inspect a website's XPOC manifest.
  • Content verifier: verify if a resource is listed in a XPOC manifest.
